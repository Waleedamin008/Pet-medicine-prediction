# Pet-medicine-prediction
Project Report: Veterinary Medicine Prediction for Pets  
1. Project Overview This project aims to automate the recommendation of target medicines for pets based on their species, age, and reported symptoms. By leveraging machine learning, we can provide data-driven insights to assist in veterinary diagnostic processes. 
2. Dataset Description The dataset contains 500 records of pet health cases with the following key features: - **Pet_Type**: The species of the pet (e.g., Dog, Cow, Horse, Rabbit). - **Pet_Age_Years**: The age of the pet. - **Disease_Reported_By_Customer**: The primary symptom or condition reported. - **Target_Medicine**: The medicine prescribed (Target Variable). - **Additional Info**: Price, Seller Rating, Country, and Delivery Status. 
3. Exploratory Data Analysis (EDA) - **Missing Values**: The dataset was clean with no missing values. - **Distributions**: We analyzed the distribution of pet types and common diseases. - **Correlation**: A heatmap was generated for numerical features (Age, Price, Rating) to understand underlying relationships. 
4. Methodology ### Data Preprocessing - **Encoding**: Categorical variables (Pet_Type, Disease_Reported_By_Customer, and Target_Medicine) were converted to numerical format using LabelEncoder. - **Splitting**: The data was split into training (80%) and testing (20%) sets. - **Scaling**: Features were normalized using StandardScaler to ensure optimal performance for distance-based models like KNN and SVM. ### Machine Learning Models Three classifiers were implemented and compared: 1. **Support Vector Machine (SVM)** with probability estimates. 2. **K-Nearest Neighbors (KNN)** with k=5. 3. **Random Forest Classifier**. 
5. Performance Evaluation The models were evaluated based on Accuracy, Confusion Matrices, and ROC Curves. ### Accuracy Comparison | Model | Accuracy | | :--- | :--- | | SVM | 46% | | KNN | 44% | | **Random Forest** | **78%** | ### Key Findings - **Random Forest** significantly outperformed other models, achieving an accuracy of 78%. - The Classification Report for Random Forest showed high precision and recall for most medicine classes, making it the most reliable model for this task. - SVM and KNN struggled with the multi-class nature of the target variable, likely due to the non-linear complexity of the symptom-medicine mapping. 
6. Conclusion The project successfully demonstrates that pet medication can be predicted with reasonable accuracy using Random Forest. Future improvements could include feature engineering (e.g., grouping symptoms) or hyperparameter tuning to further increase the predictive power.
